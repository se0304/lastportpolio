2023.10.04 ~ 2023.10.20




2023년 10월 4일부터 2023년 10월 20일까지 16일간 React를 활용하여 진행한 프로젝트 입니다.
React 외에도 React-Bootstrap, redux, scss를 활용하여 작업하였습니다.
프로젝트 속 페이지는 Intro Page, Main Page, Sub Page, Cart Page로 총 네 개의 페이지로 제작하였으며
SwiperSlide 또는 그 외 슬라이드 라이브러리를 이용하여 필요한 기능을 구현할 수 있고 리덕스를 이용한 상태 관리가 가능합니다.


SwiperSlide 라이브러리를 이용하여 메인 페이지 상단 이미지 슬라이드 기능을 구현하였습니다.

메인 페이지에서 상품 이미지 클릭 시 상세 페이지로 이동하도록 설계하였으며 장바구니 버튼 클릭시 팝업이 뜨도록 하였습니다.
또한 팝업 창에 장바구니에 담긴 상품을 바로 확인 할 수 있는 버튼과 팝업을 닫을 수 있는 버튼을 나란히 배치하였고
버튼 클릭 시 장바구니 페이지로 이동하거나 팝업을 닫도록 설계하였습니다.

장바구니 페이지에서는 상품의 갯수를 자유롭게 변경할 수 있도록 하였고 상품 갯수에 따른 상품의 가격을 함께 변경되도록 하였습니다.
또한 고객이 보다 쉽게 전체 가격 및 배송비를 확인 할 수 있도록 오른쪽에 바로 상품의 전체 금액 내역을 배치하였습니다.
